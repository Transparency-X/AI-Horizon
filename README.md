## **📌 Overview**
**AI Horizon** is an open-source platform designed to **aggregate, organize, and visualize** global in-person and virtual **AI, LLM, and ML conferences, meetups, hackathons, and workshops**. Whether you're a researcher, developer, entrepreneur, or enthusiast, AI Horizon helps you:
- **Discover** upcoming events by **region, date, focus area (e.g., LLMs, MLOps, Ethics), or cost**.
- **Filter** events by **in-person/virtual, beginner/advanced, or industry-specific** (e.g., healthcare, finance).
- **Contribute** missing events via **community submissions**.
- **Track** your favorite events with **personalized calendars and reminders**.
- **Explore trends** in AI/ML conferences with **interactive visualizations**.

🔗 **Live Demo**: [ai-horizon.live](https://ai-horizon.live) *(Coming Soon!)*

## **✨ Features**

### **🔍 Core Features**
| Feature | Description |
|---------|-------------|
| **Global Event Database** | Curated list of **500+ AI/ML/LLM conferences** (2026–2027) with **regional filters** (North America, Europe, Asia-Pacific, etc.). |
| **Advanced Search** | Filter by **date, location, focus area (LLMs, CV, NLP, MLOps), cost, or attendance size**. |
| **Interactive Map** | Visualize events on a **world map** with **clustered markers** for hotspots (e.g., San Francisco, London, Singapore). |
| **Event Details** | Each event includes **dates, location, cost, speakers, agenda, and registration links**. |
| **Personalized Calendar** | Sync events to **Google Calendar, Outlook, or iCal** and set **reminders**. |
| **Community Submissions** | Users can **submit new events** or **flag outdated ones** for moderation. |
| **Trend Analytics** | Track **popular topics (e.g., Agentic AI, RAG, Ethics)** and **attendance growth** over time. |
| **API Access** | RESTful API for **developers to integrate event data** into their own tools. |
| **Dark/Light Mode** | Accessible UI with **customizable themes**. |
| **Offline Mode** | Download event data for **offline access** (PWA support). |

### **📊 Data Coverage**
- **200+ In-Person Conferences** (e.g., NeurIPS, ICML, CVPR, AI4, NVIDIA GTC).
- **150+ Virtual/Hybrid Events** (e.g., Radar: Hybrid AI Teams, Google Cloud Next).
- **100+ Meetups & Workshops** (e.g., AgentCon, MLcon, AI Engineer World’s Fair).
- **50+ Hackathons & Competitions** (e.g., Kaggle, AI Hackathons).

## **🗺️ Roadmap**
*Our vision for **AI Horizon** in the coming months.*

### **🚀 Short-Term (Q2–Q3 2026)**
- [x] **Launch MVP** with **500+ events** (2026–2027).
- [x] **Interactive map** with **filtering by region/focus area**.
- [x] **Community submission portal** for new events.
- [ ] **API v1.0** for developers (RESTful endpoints).
- [ ] **Mobile app** (iOS/Android) with **offline mode**.
- [ ] **Browser extension** to **track events from any webpage**.

### **🌱 Mid-Term (Q4 2026–Q1 2027)**
- [ ] **AI-Powered Recommendations**: Suggest events based on **user interests** (e.g., LLMs, MLOps).
- [ ] **Speaker/Attendee Networking**: Connect with **other participants** before/after events.
- [ ] **Event Reviews & Ratings**: Community-driven **feedback system**.
- [ ] **Integration with Calendly/Zoom**: Auto-schedule **1:1 meetups** at conferences.
- [ ] **Scholarship Tracker**: List **grants, discounts, and diversity programs** for events.

### **🌌 Long-Term (2027+)**
- [ ] **AI Horizon Pro**: **Premium features** (e.g., **exclusive event invites, early-bird discounts**).
- [ ] **Virtual Event Hosting**: Allow organizations to **host hybrid events** on the platform.
- [ ] **AI-Powered Summaries**: Generate **automated recaps** of conference talks.
- [ ] **Global Hackathon Hub**: Aggregate **AI/ML hackathons** with **team-matching**.
- [ ] **Blockchain Verification**: **Decentralized verification** of event attendance (for certifications).
